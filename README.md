## ğŸ“š Learning Path

The curriculum is divided into 4 phases over 8 weeks:

### Phase 1: Foundations (Weeks 1-2)
- Challenge 1.1: Moving Circle - Coordinates and basic movement
- Challenge 1.2: Bouncing Ball - Physics and conditionals
- Challenge 1.3: Gravity Simulator - Acceleration and time-based movement

### Phase 2: Interactive Games (Weeks 3-4)
- Challenge 2.1: Catch the Falling Numbers
- Challenge 2.2: Projectile Motion Game
- Challenge 2.3: Vector Field Visualization

### Phase 3: Advanced Mechanics (Weeks 5-6)
- Challenge 3.1: Pendulum Simulator
- Challenge 3.2: Orbit Simulator
- Challenge 3.3: Wave Interference

### Phase 4: Capstone Projects (Weeks 7-8)
- Challenge 4.1: Physics Puzzle Game
- Challenge 4.2: Math Racing Game

See [`CBL_LEARNING_PLAN.md`](CBL_LEARNING_PLAN.md) for the complete detailed roadmap.

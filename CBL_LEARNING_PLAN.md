## üéì Teaching Approach

### For Each Challenge:
1. **Introduction** (10 min): Explain the math/physics concept
2. **Demo** (5 min): Show the final result
3. **Guided Coding** (60-90 min): Code together, explain each step
4. **Experimentation** (30 min): Let Samuel modify and experiment
5. **Reflection** (10 min): Discuss what was learned

### Tips:
- Always connect code to math/physics concepts
- Let mistakes happen - debugging is learning
- Encourage experimentation and "what if" questions
- Celebrate small wins
- Build on previous challenges

## Tips for Learning

1. **Run the code first** - See what it does
2. **Read the comments** - They explain what's happening
3. **Experiment** - Change values and see what happens
4. **Answer reflection questions** - Think about what you learned
5. **Try the experiment ideas** - Extend the code yourself!
